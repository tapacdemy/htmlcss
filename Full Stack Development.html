Datatypes

An SQL developer must decide what type of data that will be stored inside each column when creating a table. The data type is a guideline for SQL to understand what type of data is expected inside of each column, and it also identifies how SQL will interact with the stored data.

Constraints

Constraints can be specified when the table is created with the CREATE TABLE statement, or after the table is created with the ALTER TABLE statement.

Introduction to Html

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content.

Tags

An HTML element is defined by a start tag, some content, and an end tag.HTML headings are titles or subtitles that you want to display on a webpage.A paragraph always starts on a new line, and is usually a block of text.

Attributes

All HTML elements can have attributes Attributes provide additional information about elements Attributes are always specified in the start tag Attributes usually come in name/value pairs like: name="value"

Forms

An HTML form is used to collect user input. The user input is most often sent to a server for processing.

Tables

HTML tables allow web developers to arrange data into rows and columns.

Lists

HTML lists allow web developers to group a set of related items in lists.An unordered list starts with the ul tag. Each list item starts with the li tag.

Links

HTML links are hyperlinks. You can click on a link and jump to another document. When you move the mouse over a link, the mouse arrow will turn into a little hand.

Entities

Some characters are reserved in HTML. If you use the less than (<) or greater than (>) signs in your HTML text, the browser might mix them with tags. Entity names or entity numbers can be used to display reserved HTML characters.
